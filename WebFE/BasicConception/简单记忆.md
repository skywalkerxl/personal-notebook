# 基本数据类型简单记忆

### 1. 简单数据类型 UNSNB

1. Undefined
2. Null
3. Boolean
4. Number
5. String

### 2. 复杂数据类型 O

1. Object

### 3. 含义及说明

| 类型      | 含义   | 说明                                                         |
| --------- | ------ | ------------------------------------------------------------ |
| Undefined | 未定义 | 该类型只有一个值，即特殊的undefined，声明却未初始化的值即为undefined |
| Number    | 数值   |                                                              |
| String    | 字符串 |                                                              |
| Null      | 布尔值 | 该类型是第二个只有一个值的数据类型，即null,表示一个空对象指针 |
| Boolean   | 函数   |                                                              |
| Object    | 对象   |                                                              |
